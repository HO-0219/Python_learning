패키지와 라이브러리 정리

설치 요약

현재 Day1~Day9 코드에서 pip 설치가 필요한 외부 패키지는 아래 2개이다.

pip install requests aiohttp

또는 requirements.txt를 사용한다.

pip install -r requirements.txt

표준 라이브러리는 Python에 기본 포함되어 있으므로 pip install이 필요 없다.
대신 코드에서 import만 하면 된다.

1. 외부 패키지

requests
- 설치 필요: pip install requests
- 사용 파일:
  - Day9/HTTPTest.py
  - Day9/http_b_nb.py
- 용도: HTTP GET, POST 요청을 동기 방식으로 보낸다.
- import:
  import requests

aiohttp
- 설치 필요: pip install aiohttp
- 사용 파일:
  - Day9/Asyncio_HTTP_test.py
  - Day9/http_b_nb.py
- 용도: HTTP 요청을 비동기 방식으로 보낸다.
- import:
  import aiohttp

5. 수업 PDF 후반부를 실습할 때 추가로 필요할 수 있는 패키지

PDF에는 Pandas, 시각화, 공공 데이터, 크롤링 관련 내용도 포함되어 있다.
이 부분까지 실습하려면 아래 패키지가 필요할 수 있다.

pip install pandas matplotlib seaborn koreanize-matplotlib openpyxl requests beautifulsoup4 lxml

용도
- pandas: 표 형태 데이터 분석
- matplotlib: 기본 시각화
- seaborn: 통계 기반 시각화
- koreanize-matplotlib: 그래프 한글 표시
- openpyxl: Excel 파일 읽기/쓰기
- requests: HTTP 요청
- beautifulsoup4: HTML 파싱
- lxml: XML/HTML 파싱 보조
