웹 프레임워크, 프론트엔드, 배포, 인증, 개인 메모, 설치 파일, 압축 백업, 캐시 파일처럼 Python 문법 학습과 직접 관련이 낮은 자료는 제거했습니다.

## 보는 순서

1. `Ict-Python-수업정리.txt`로 수업 PDF의 핵심을 먼저 확인합니다.
2. `Python학습 과정`으로 전체 학습 흐름을 봅니다.
3. `패키지_라이브러리_정리.txt`에서 필요한 import와 pip 설치 항목을 확인합니다.
4. 각 `Day*/n회차` 메모로 그날의 개념을 훑습니다.
5. 같은 폴더의 `.py` 파일을 실행하며 예제를 확인합니다.

## 정리 기준

- 순수 Python 문법, 표준 라이브러리, 알고리즘, 네트워크 예제는 유지했습니다.
- 수업 PDF는 Python 수업 원본이므로 유지했습니다.
- 웹 프레임워크, 프론트엔드, 배포, 인증, 개인 메모 문서는 제거했습니다.
- `.DS_Store`, `__pycache__`, `.pyc`, `.zip`, `.dmg`, `.pkg`, 비Python `.pdf`/`.hwp` 같은 부산물은 제거했습니다.
