5. 리스트 주요 함수

- append(): 리스트 끝에 요소 추가
- insert(): 원하는 위치에 요소 추가
- extend(): 다른 리스트를 뒤에 이어 붙임
- index(): 특정 값의 위치 반환
- pop(): 특정 위치의 요소를 꺼내고 삭제
- remove(): 특정 값을 삭제
- clear(): 모든 요소 삭제
- count(): 특정 값의 개수 반환
- sort(): 리스트 정렬

8. 조건문

조건문은 조건에 따라 실행할 코드를 나누는 문법이다.
Python에서는 블록을 만들 때 콜론(:)과 들여쓰기를 사용한다.

if 조건:
    실행할 코드
elif 다른조건:
    실행할 코드
else:
    실행할 코드

9. 비교 연산자와 논리 연산자

- a > b: a가 b보다 크다.
- a < b: a가 b보다 작다.
- a >= b: a가 b보다 크거나 같다.
- a <= b: a가 b보다 작거나 같다.
- a == b: a와 b가 같다.
- a != b: a와 b가 다르다.
- and: 조건이 모두 참일 때 참
- or: 조건 중 하나라도 참이면 참
- not: 참과 거짓을 반대로 바꿈

10. 반복문

for문은 정해진 범위를 반복할 때 사용한다.

for i in range(5):
    print(i)

while문은 조건이 참인 동안 반복한다.

count = 0
while count < 5:
    print(count)
    count += 1

주의
while문은 조건이 계속 참이면 무한 반복될 수 있다.
필요할 때 break로 반복을 종료한다.
