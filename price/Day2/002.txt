1. 리스트 컴프리헨션

리스트 컴프리헨션은 리스트를 짧고 읽기 쉽게 만드는 문법이다.

기본 형태
새리스트 = [표현식 for 변수 in 반복가능한값 if 조건]

예시
squares = [x ** 2 for x in range(1, 6)]
evens = [x for x in range(1, 11) if x % 2 == 0]

2. lambda 함수

lambda는 이름 없는 짧은 함수를 만들 때 사용한다.
복잡한 로직보다는 한 줄짜리 간단한 계산에 적합하다.

예시
add = lambda a, b: a + b
print(add(3, 5))

3. 자주 쓰는 내장 함수

map(함수, 반복가능한값)
- 각 요소에 함수를 적용한다.

filter(함수, 반복가능한값)
- 조건을 만족하는 요소만 남긴다.

zip(a, b)
- 여러 리스트를 같은 위치끼리 묶는다.

enumerate(반복가능한값)
- 인덱스와 값을 함께 꺼낸다.

sorted(반복가능한값, key=함수, reverse=True/False)
- 정렬된 새 결과를 반환한다.

자주 쓰는 기본 함수
- len(): 길이
- sum(): 합계
- max(): 최댓값
- min(): 최솟값
- abs(): 절댓값
- round(): 반올림
- type(): 자료형 확인
- input(): 사용자 입력
- int(), float(), str(): 형변환

4. 모듈

모듈은 재사용할 코드를 담은 .py 파일이다.
다른 파일에서 import로 불러와 사용할 수 있다.

기본 사용
import math
print(math.sqrt(16))

특정 함수만 가져오기
from math import sqrt
print(sqrt(16))

별칭 사용
import math as m
print(m.sqrt(16))

직접 실행될 때만 실행하기
if __name__ == "__main__":
    print("이 파일을 직접 실행할 때만 실행됩니다.")

5. 패키지

패키지는 여러 모듈을 폴더 단위로 묶어 관리하는 구조이다.
폴더 안에 __init__.py 파일을 두면 패키지로 인식된다.

예시 구조
my_package/
    __init__.py
    calculator.py

사용 예시
from my_package import calculator
print(calculator.add(1, 2))

6. 예외 처리

예외 처리는 오류가 발생해도 프로그램이 바로 종료되지 않도록 처리하는 방법이다.

try:
    number = int(input("숫자 입력: "))
    print(10 / number)
except ValueError:
    print("숫자를 입력해야 합니다.")
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")

핵심
- try에는 오류가 날 수 있는 코드를 작성한다.
- except에는 오류가 났을 때 실행할 코드를 작성한다.
- finally는 오류 발생 여부와 상관없이 실행된다.
